개발 환경 : 구름 ide 
### 설치
 - 구름 ide 의경우 컨테이너 생성시 mysql 설치를 선택 하여 설치 가능 
 `sudo apt-get install -y mysql-server` 명령어로 설치 할수 있다 
### 접속 
`mysql -u root -p`로 mysql 실행이 가능 하나 root 비밀 번호를 설정해야 합니다.
