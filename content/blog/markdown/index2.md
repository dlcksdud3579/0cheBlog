# 문제 
 - 크롬 앱에서 실행시 깃허브 인증후 stackedit.io로 넘어오면서 진행 되지 않는다.
 - 크롬 시크릿 탭으로 실행한 경우 인증 정보가 넘어 오며 잘 동작하는 것을 확인
 - 자네이버 웨일에서 또한 정상 작동한다.

#  문제 해결
 1. 크롬 데이터 삭제 와 모든 캐시를 삭제한다. 
 2. stackedit.io 를 크롬 어플로 바탕화면의 설치
 3. 크롬을 기본 브라우저로 설치
 4. 깃허브랑 연관된 모든 어플을 제거 (저는 ㄹ)
